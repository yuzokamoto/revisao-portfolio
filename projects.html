Pokedex - Front End

Pokedex é um site de pokémons, onde o usuário pode capturar os pokemons para sua própria Pokedex, e visualizar detalhes de cada um deles e é composto por 3 Pages: Home, Pokedex e Detalhes.
Este projeto tem como fonte de dados para a sua criação a Poke Api, uma Api pública, muito usada para aplicações focadas em aprendizado de programação e também usada em cases de processos seletivos.

Tecnologias: HTML, React, JavaScript, Styled-components, React Router e Integração com API.

To do List - Backend

To do List foi desenvolvido para ser um back-end para uma aplicação de gestão de tarefas de uma empresa, onde deve ser possível cadastrar membros e tarefas da equipe.
Membros só precisarão de nome, e-mail e senha, enquanto tarefas terão título, descrição, data de criação e seu status se está feita ou não. Cada tarefa inicia sem responsável, mas também deve ser possível atribuir mais de uma pessoa responsável para a mesma tarefa. Não há autenticação de usuário.

Tecnologias: NodeJs, TypeScript, Express, Criação de API REST, Banco de Dados MySql, Knex e Postman.
